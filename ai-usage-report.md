# AI Usage Report

## Tools Used

- ChatGPT

## How AI Was Used

ChatGPT was used as a support tool during the development of this assignment.

It helped with:
- suggesting ideas for advanced portfolio features
- reviewing the structure of the HTML, CSS, and JavaScript files
- explaining how to connect the website to an external API
- helping improve filtering, sorting, and validation logic
- reviewing documentation structure and wording

## Benefits and Challenges

Using AI helped speed up the development process and made it easier to understand how to organize the project features in a clearer way.

It was also useful for checking logic and improving the structure of the code.

One challenge was that suggested code or ideas still needed review and adjustment to fit the project requirements correctly.

## Learning Outcomes

Through this assignment, I learned:
- how to fetch and display data from an external API
- how to filter and sort dynamic data
- how to manage simple application state using localStorage
- how to improve form validation with multiple checks
- how to review and adapt AI suggestions instead of using them without understanding

## Responsible Use and Modifications

AI was used as a learning and support tool only.

The suggestions were reviewed, tested, and adjusted before being included in the final version of the project.

I modified the structure, wording, and logic where needed so the final result matched the assignment requirements and worked correctly inside my portfolio website.
