# AI Usage Report

## Tools Used

- ChatGPT

## How AI Was Used

ChatGPT was used as a support tool during the development of this assignment.

It helped with:
- suggesting ways to improve the final portfolio structure
- reviewing and refining the HTML, CSS, and JavaScript organization
- improving the user interface and section layout
- checking ideas for repository search, filtering, and sorting
- reviewing form validation logic and user feedback messages
- improving the wording and organization of the documentation

## Benefits and Challenges

Using AI helped speed up the process of reviewing the project and improving its structure.

It was useful for exploring better ways to organize features, polish the layout, and present the application more professionally.

One challenge was that suggested solutions still required testing and adjustment to fit the project correctly and match the assignment expectations.

## Learning Outcomes

Through this assignment, I learned:
- how to refine a web application into a more polished final product
- how to improve user experience through layout and interaction design
- how to organize multiple features into one complete application
- how to review and adapt technical suggestions before applying them
- how to use AI as a support tool for both coding and documentation

## Responsible Use and Modifications

AI was used as a learning and support tool only.

The suggestions were reviewed, tested, and adjusted before being included in the final version of the project.

I modified the structure, wording, and implementation details where needed so the final result matched the assignment requirements and worked correctly in my portfolio website.
